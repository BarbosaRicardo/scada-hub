{\small\sffamily\color{igray}\MakeUppercase{Course Topics}}
\smallskip

\setlist[enumerate]{noitemsep, topsep=2pt, leftmargin=*, label=\textcolor{iblue}{\small\bfseries\arabic*.}}
\begin{enumerate}\small
  \item \textbf{Introduction:} The pre-standard PLC landscape, what the five-language specification unified, and what it deliberately left vendor-specific.
  \item \textbf{Data Types:} BOOL, INT, DINT, REAL, TIME, STRING, and user-defined structures; type mismatch consequences in live systems.
  \item \textbf{Structured Text:} Pascal-style syntax — IF/THEN, FOR, WHILE, CASE — and appropriate use cases among the five languages.
  \item \textbf{Ladder Diagram:} Contact/coil notation, rung structure, and reading existing automation code in the industry's most common language.
  \item \textbf{Function Block Diagram:} Data-flow graphical programming, block connectivity, and use cases in PID and signal conditioning.
  \item \textbf{Sequential Function Chart:} Steps, transitions, and action qualifiers for modeling processes with discrete ordered states.
  \item \textbf{Program Organization Units:} Functions, Function Blocks, Programs, and Configurations; reusable library design across vendors.
  \item \textbf{RTAC Integration:} IEC 61131-3 in ACSELERATOR Architect and translation from PLC programming to substation automation logic.
  \item \textbf{Troubleshooting:} Implicit type conversion, scan cycle overruns, latching coil failures, and silent runtime faults.
  \item \textbf{Practice Lab:} PID controller in Structured Text, motor starter in Ladder, batch sequence in SFC, analog scaling Function Block.
\end{enumerate}

{\small\sffamily\color{igray}\MakeUppercase{Learning Outcomes}}
\smallskip

\setlist[itemize]{noitemsep, topsep=2pt, leftmargin=*, label=\textcolor{iblue}{\small$\bullet$}}
\begin{itemize}\small
  \item Write and debug programs in Structured Text, Ladder Diagram, and Sequential Function Chart.
  \item Design reusable Function Blocks with typed interfaces for use across multiple vendor platforms.
  \item Interpret and modify existing Ladder Diagram code in a production automation system.
  \item Configure a PID controller using the standard IEC 61131-3 function block interface.
  \item Diagnose scan cycle overruns, type conversion errors, and silent runtime faults in live PLC programs.
\end{itemize}
