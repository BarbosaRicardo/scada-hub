{\small\sffamily\color{sgray}\MakeUppercase{Overview}}
\smallskip

{\small 8 study guide sites covering Modbus, OPC UA, DNP3, IEC 61131-3, PID Controllers, SEL RTAC, Ignition SCADA, and Wireshark. Each guide is a React/Vite SPA deployed to GitHub Pages, with quizzes, flashcards, a Code Lab, and a downloadable course PDF. The hub landing page (scada-hub.vercel.app) links all 8 guides.}

  \item \textbf{Auth: Duplicate Email Signup Error} \hfill \med \\
    Registering with an existing email shows a generic Supabase error. Show a clear "account exists" message with a "Reset password" link. Add reset-password flow triggered from the register form error state. Scope: scada-hub \texttt{AuthPage.jsx}.

  \item \textbf{Quiz Reports RLS — Supabase} \hfill \med \\
    Any logged-in user can currently read any other user's quiz history. Add Supabase row-level security (RLS) policies on the \texttt{quiz\_results} table: users see only their own records; admin account has a bypass policy.

  \item \textbf{Mobile UX Audit} \hfill \med \\
    Overall mobile layout is unfriendly across all 8 guides and the hub. Full audit needed: text sizes, tap targets, sidebar behavior, LearningPath horizontal scroll on small screens.

  \item \textbf{Supabase Email Confirmation Redirect} \hfill \low \\

  \item \textbf{LearningPath Hover Scale Bug} \hfill \low \\
    Hovering a step card triggers \texttt{whileHover=\{scale: 1.03\}} — card enlarges and text becomes hard to read. Reduce or remove the scale transform. Scope: \texttt{scada-hub/site/src/components/LearningPath.jsx}.

  \item \textbf{RTAC PDF Layout Overhaul} \hfill \low \\
    RTAC study guide PDF has bad spacing, orphaned headings, and an intro style inconsistent with the Modbus PDF template. Needs a formatting pass to align with the Modbus LaTeX template.

    Current OG images use Python PIL with the system Helvetica font — functional but basic. Replace with a custom SVG design or Vercel's \texttt{@vercel/og} for server-side image generation at deploy time. Scope: scada-hub \texttt{public/og-image.png} + all 8 guide repos.
